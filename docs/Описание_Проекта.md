Предметная область:
	 Веб-сервис «Зеленый Щит» предоставляет услуги по консультации и установке умных систем для клиентов, чтобы помочь им снизить затраты воды и электричества в частных домах, квартирах и других помещениях, задействованных как предприятие для малого бизнеса.  Также сайт включает в себя интернет-магазин с каталогом, корзиной, оплатой и возможностью доставки, установки или сохранения на отдельном складе до 30 дней, если пользователь захотел сам забрать то или иное устройство. Этот проект разрабатывается с использованием DevOps-подходов, чтобы обеспечить полноценную автоматизацию, тестирование и надёжный деплой. 

Описание проекта:
	«Зелёный Щит» — это полноценное веб-приложение, которое сочетает в себе витрину услуг и интернет магазин. Основные функции:
    • Каталог карточек товаров (умные устройства с подробным описанием и ценами)
    • Корзина и оформление заказа с выбором доставки/установки/сохранности на складе(если клиент отказался от доставки).
    • Онлайн-калькулятор для удобного расчёта экономии воды и электричества
    • Форма заявки для записи на выездную консультацию
    • Личный кабинет для пользователей, где они могут видеть аналитику по экономии
    • Чат для общения с консультантами

Архитектура проекта:
	Теперь нужно выбрать архитектуру для проекта и между монолитной и микросервисной  было принято решение принять сторону второй архитектуры, где каждая функция сайта реализована как отдельный сервис, взаимодействующий через REST API. Это позволяет обновлять функции без изменения всего сайта и добавлять новые для улучшения UX. 

Основные части проекта:
Фронтенд будет реализован в виде интерфейса на русском языке с адаптивным дизайном, включающий в себя страницы с каталогом, корзиной, калькулятором, личным кабинетом, чатом и др.
Бэкенд же будет включать в себя следующие микросервисы:
    • Фронтенд, который отображает все страницы сайта.
    • Каталог товаров, показывающий устройства с описанием, ценами, поиском и фильтрами.
    • Корзина, управляющая списком выбранных товаров
    • Заказы, обрабатывающие оформление заказов, включая доставку и установку. 
    • Оплата, интегрируемая вместе с платёжными системами для транзакций.
    • Калькулятор, рассчитывающий экономию на основе данных пользователя.
    • Заявки, обрабатывающие формы консультации.
    • Авторизация, управляющая доступом к личному кабинету.
    • Аналитика, генерирующая отчеты по экономии.
    • Чат, обеспечивающий общение с консультантами. 
Будут реализованы несколько баз данных, которые будут хранить данные пользователей, заказов, заявок, аналитики, каталога.
Также будут полноценные интеграции с платёжными системами, например такие как, СБП или Юkassa, сервисами доставки, такие как СДЭК или Boxberry.
AmoCRM или Bitrix24 для обработки заявок, SendGrid для email-уведомлений и Google Analytics для анализа трафика. Все эти интеграции идут как пример и они вполне могут измениться в ближайшем будущем.


Pipleline проекта:
Разработка проекта организована с иcпользованием DevOps для автоматизации и качества работы проекта. Сам процесс будет работать следующим образом:
    1. Код хранится в репозитории на GitHub.
    2. Разработчик создаёт ветку( для примера возьмём feature/orders) для новой функции.
    3. Делает коммиты через терминал VSCode.
    4. Открывает pull request в ветку dev
    5. GitHub Actions запускает тесты (проверка кода, функциональности корзины, оплаты)
    6. После review изменения сливаются в dev.
    7. Ветка dev сливается в main.
    8. GitHub Actions деплоит микросервисы в Docker-контейнерах на AWS/Vercel
Технологии проекта:
    • React используется для интерактивного интерфейса на русском языке .
    • Tailwind CSS для адаптивного дизайна.
    • Node.js + Express для работы микросервисов .
    • MongoDB для хранения данных пользователей, заказов, заявок, аналитики, каталога.
    • Docker для упаковки микросервисов.
    • GitHub Actions для автоматизации тестов и деплоя.
    • AWS/Vercel для хостинга.
    • ЮKassa/Тинькофф Оплата для обработки платежей картами, СБП.
    • СДЭК/Boxberry API для расчёта доставки.
    • AmoCRM/Bitrix24 для обработки заявок и заказов.
    • SendGrid для email-уведомлений.
    • Google Analytics для анализа трафика.

Заключение
Веб-сайт «Зеленый Щит» сочетает в себе витрину услуг умного дома и интернет-магазин. Микроервисная архитектура делает сайт гибким, а интеграции с Юkassa и СДЭК упрощают оплату и доставку. DevOps-инструменты, такие как GitHub, Docker, GitHub Actions автоматизируют разработку и деплой, обеспечивая надёжность.
